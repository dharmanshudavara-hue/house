#  Interactive Spline Landing Page

A modern **interactive landing page** built using **React + Vite** and a **Spline 3D scene**.
The page supports smooth scrolling, mouse interaction, and responsive design.

##  Features
- Interactive 3D Spline model
- Hero section with 3D focus
- Scroll-based content sections
- Mouse / hover interaction
- Responsive layout
- Vercel-ready deployment

##  Tech Stack
- React 18
- Vite
- @splinetool/react-spline
- HTML, CSS, JavaScript
